<h1 align="center">🛡️ SafetyCheck</h1> 
<h3 align="center">AI-Powered PPE Detection for Healthcare Safety</h3>

SafetyCheck is a deep learning-based system designed to ensure healthcare workers and visitors wear appropriate Personal Protective Equipment (PPE) before entering patient areas. Using real-time camera feeds and a CNN model, it detects missing gear and helps maintain a safe hospital environment.
